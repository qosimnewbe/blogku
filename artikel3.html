Mengenal JavaScript

25 Juli 2025

CJavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif.

Fungsi JS(JavaScript)

  • Fungsi Utama: Menambahkan dinamisme ke website (seperti animasi, form validasi, update konten tanpa reload).
  • Berjalan di Browser: Dieksekusi di sisi klien (client-side), tetapi kini juga bisa di server (Node.js).
  • Sintaks Mudah: Mirip dengan bahasa seperti C++ atau Java, tetapi lebih fleksibel.
  • Fitur Kunci:
  • Manipulasi DOM (mengubah elemen HTML/CSS).
  • Event handling (respons terhadap klik, input, dll.).
  • Async programming (AJAX/fetch untuk komunikasi dengan server).

Contoh JS Sederhana


                document.getElementById("tombol").addEventListener("click", function() {
                    alert("Hello World!");
                });

JavaScript esensial untuk pengembangan web modern, bekerja bersama HTML (struktur) dan CSS (tampilan).
