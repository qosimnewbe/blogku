Mengenal CSS dan Styling

25 Juli 2025

CSS (Cascading Style Sheets) adalah bahasa desain yang digunakan untuk memperindah tampilan HTML. Tanpa CSS, website hanya berupa teks polos dan sangat sederhana.

Manfaat CSS

  • Mempercantik tampilan website
  • Mengatur layout dan tata letak
  • Menyesuaikan desain untuk berbagai ukuran layar (responsive)

Contoh CSS Sederhana

p {
                color: blue;
                font-size: 16px;
            }

Dengan beberapa baris CSS, website kamu langsung berubah menjadi lebih menarik!
